# ERP Project — Claude Instructions

## Проект

Веб-ориентированная ERP-система для предприятий общественного питания (ресторан / кафе).

**Стек:** Node.js + Hono (backend), React + MUI (frontend), PostgreSQL, Docker, Caddy.

**Автор:** Roman

---

## Дипломная работа

**Тема:** Разработка веб-ориентированной ERP-системы для предприятий общественного питания

**Цель:** автоматизация управления бронированием, заказами, складом, персоналом и финансовой аналитикой ресторана/кафе.

Файлы диплома находятся в папке `diploma/`. При работе с ними соблюдай правила ниже.

### Стиль написания (для файлов в `diploma/`)

- **Язык:** русский, академический стиль
- **Тон:** нейтральный, безличный («в ходе исследования было установлено...», «разработанная система позволяет...»)
- **Избегать:** разговорных оборотов, местоимения «я», сокращений без расшифровки
- **Термины:** при первом упоминании давать расшифровку (ERP — Enterprise Resource Planning)
- **Объём абзаца:** 6–10 предложений, логически завершённый
- **Объём раздела:** писать максимально развёрнуто — каждый подраздел минимум 3–5 абзацев; не сокращать без явной просьбы
- **Детализация:** раскрывать каждый тезис подробно: давать определение понятия, описывать его характеристики, приводить примеры применения в контексте проекта, делать вывод
- **Связность:** между абзацами использовать переходные фразы («Следует отметить...», «Наряду с этим...», «Таким образом...», «В рамках данной работы...»)
